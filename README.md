# Portfolio React - Anthony Porquet

Ce projet est un portfolio personnel développé avec **React.js** et **Bootstrap**.  
Il présente mes services, mes réalisations, un formulaire de contact ainsi que des mentions légales.

---

## 🚀 Technologies utilisées

- React.js
- Vite
- Bootstrap 5
- CSS3
- React Router DOM
- React Helmet Async

---

## 📦 Prérequis

Avant de lancer le projet, vous devez installer :

- Node.js (version 16 ou supérieure)
- npm ou yarn
